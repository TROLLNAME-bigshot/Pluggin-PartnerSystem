# Pluggin-PartnerSystem

Система партнёрских промокодов для Minecraft-сервера.

## Описание

**Pluggin-PartnerSystem** — это микросервис и игровой плагин, позволяющий организовать партнёрскую программу для стримеров и контент-мейкеров.

Стримеры получают уникальные промокоды. Игроки активируют их на сервере и совершают покупки доната на сайте. После этого информация о продажах синхронизируется с игровым сервером, а стример может отслеживать свою статистику и доход прямо в игре.

## Основные возможности

* Создание и управление партнёрскими промокодами.
* Привязка промокодов к стримерам.
* Учёт покупок доната по каждому промокоду.
* Автоматический расчёт вознаграждений.
* Просмотр статистики в игре.
* RCOM для взаимодействия сайта и игрового сервера.
* Система логирования действий и транзакций.
* Защита от дублирования покупок.

## Сценарий работы

1. Администратор создаёт партнёрский промокод.
2. Промокод выдаётся стримеру.
3. Игрок активирует промокод на сервере.
4. Игрок покупает донат на сайте.
5. Сайт отправляет информацию о покупке в микросервис.
6. Система фиксирует доход стримера.
7. Стример может посмотреть статистику через игровые команды.

## Команды

| Команда         | Описание                  |
| --------------- | ------------------------- |
| /promos  | Показать статистику партнёров    |
| /mypromo  | Показать свою статистику        |
| /partner income | Показать доход            |
| /promocheck   | Информация о промокоде      |
| /promotop    | Топ партнёров                |


## Цель проекта

Разработка системы партнёрских промокодов для Minecraft-серверов с возможностью отслеживания доходов стримеров в режиме реального времени.
